Ez-Map Introduction

TESTING THE VERSIONING:

This should always appear.

Welome to EZ-Map - a toolkit for mapping environments using LiDAR, setting up autonomous routes, or manaully driving a robot with video streaming and remote control.

Setup

Before you begin, make sure you have all the required hardware and components ready.

Magni Robot

To run EZ-Map, you need a Magni robot. If you’re new to the platform, start with the Start Here guide.

Choose your robot model and follow the relevant Getting Started guides and the Full Assembly guides.

LiDAR Sensor

A mounted LiDAR is required for mapping. EZ-Map currently supports the following models out of the box:

  • N10 - Short Range LiDAR

  • MS1L - Long Range LiDAR

Other LiDARs may work if the appropriate ROS driver is installed. For unsupported models, contact Ubiquity Robotics support.

Note

This documentation assumes you’re using a clean Ez-Map image. Disregard any manual configuration steps mentioned in other setup guides.

Pi Camera

The Pi Camera provides video streaming and fiducial marker detection. It should come pre-mounted on Magni 6 Robots.

If your camera is not installed, refer to the assembly guides for your specific robot model.

EZ-Map Software Image

You will receive the Ez-Map img.xz software image by email. After downloading, follow the guide to flsh it to a microSD card.

To verify the version you are using, refer to the Changelog.

How to use EZ-MAP

Once setup is complete and the robot has booted, you’re ready to start using Ez-Map.
